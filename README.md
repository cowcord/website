# My Website

My website, built with [Qwik] and [Tailwind]

## To run

#### Prerequisites

-   Bun v1.2.19+

#### Steps

-   Clone repository

```sh
git clone https://github.com/not-a-cowfr/website cow-website
cd cow-website
```

-   Install deps

```sh
bun i
```

-   Copy .env file

```sh
cp .env.example .env.local
```

To run in a development environment:

```sh
bun dev
```

[Qwik]: https://qwik.dev/
[Tailwind]: https://tailwindcss.com/
