## To run

#### Prerequisites

-   Git
-   Bun v1.2.19+

#### Steps

-   Clone repository

```sh
git clone https://github.com/cowcord/website cowcord-site
cd cowcord-site
```

-   Install deps

```sh
bun i
```

-   Copy .env file

```sh
cp .env.example .env.local
```

To run in a development environment:

```sh
bun dev
```
